## Dependencies

* python-virtualenv

## Required pip packages

* boto3
* pyspark
* fuzzywuzzy
* python-Levenshtein
* requests
* PyYAML
* pandas
* numpy
* jupyter

## How to

To start:

`virtualenv venv`

`source venv/bin/activate`

Install pip packages:

`pip install <<packages>>`

Execute:

`jupyter-notebook`
